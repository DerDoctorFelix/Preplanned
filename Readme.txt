This is a BLT friendly version of the Mod "Preplanned" by ThatGuyFromBreakingBad. The original can be found at https://github.com/thatguyfrombb/Preplanned/

You need to bind the key in the Mod Keybind menu. (Start the game, go to OPTIONS, then to MOD KEYBINDS.)

The folder "preplanned" needs to go into your "mods" folder.

Modification done by Reddit user /u/DerFelix
